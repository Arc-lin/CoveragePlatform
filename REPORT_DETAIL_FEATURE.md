# 覆盖率报告详情页功能说明

## 功能概述

已实现覆盖率报告详情页，支持：
1. 点击报告列表中的任意报告跳转到详情页
2. 显示报告的详细信息（Commit、分支、覆盖率统计）
3. 显示文件列表及每个文件的覆盖率
4. 点击文件查看代码级覆盖率，用颜色标记覆盖/未覆盖的行

## 页面结构

### 1. 报告信息卡片
- 报告 ID
- Commit Hash（完整和短格式）
- 分支信息
- 创建时间
- 覆盖率统计（行覆盖率、函数覆盖率、分支覆盖率、增量覆盖率）
- 可视化进度条

### 2. 文件列表
- 显示所有被测试的文件
- 每个文件显示覆盖率百分比
- 颜色标记：绿色(≥80%)、黄色(60-80%)、红色(<60%)
- 点击文件查看详细代码覆盖情况

### 3. 代码查看器
- 显示文件的每一行代码
- **覆盖的行**：绿色背景 + ✓ 标记
- **未覆盖的行**：红色背景 + ✗ 标记
- 显示行号
- 显示指令覆盖/未覆盖数量
- 图例说明

## 颜色标记说明

| 颜色 | 含义 | 标识 |
|------|------|------|
| 🟢 绿色 | 已覆盖 | ✓ |
| 🔴 红色 | 未覆盖 | ✗ |
| 🟡 黄色 | 部分覆盖（文件级）| 覆盖率 60-80% |

## 技术实现

### 后端 API

1. **获取文件列表**
```
GET /api/coverage/:id/files
```
返回报告中所有文件的覆盖率统计。

2. **获取文件行级覆盖率**
```
GET /api/coverage/:id/file?path=文件名
```
返回指定文件的每一行的覆盖情况。

### 前端组件

- **ReportDetail.tsx**: 详情页主组件
  - 报告信息展示
  - 文件列表
  - 代码查看器（带颜色标记）

## 使用方式

1. 访问报告列表页面：`http://localhost:3000/reports`
2. 点击任意报告的 "View Details" 按钮或表格行
3. 进入详情页查看：
   - 报告基本信息
   - 覆盖率统计
4. 在左侧文件列表点击任意文件
5. 右侧代码查看器显示：
   - 绿色行：已覆盖的代码
   - 红色行：未覆盖的代码

## 注意事项

1. 代码内容是模拟生成的（实际项目中需要从源代码仓库读取）
2. 覆盖/未覆盖标记基于 JaCoCo 的指令覆盖率数据
3. 绿色表示该行至少有一条指令被执行
4. 红色表示该行的指令完全未被执行
